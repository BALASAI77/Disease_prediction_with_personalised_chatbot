# Disease_prediction_with_personalised_chatbot
This Streamlit application predicts diseases based on symptoms selected by the user and provides diet, fitness, and home remedies suggestions for the predicted disease. Additionally, it displays a disease severity bar based on the selected symptoms.

Features
Symptom Selection: Users can select up to 5 symptoms from a dropdown menu.
Disease Prediction: The app predicts the disease based on the selected symptoms using a decision tree classifier.
Severity Bar: Displays a severity bar indicating the severity percentage of the predicted disease.
Diet, Fitness, and Home Remedies Suggestions: Provides suggestions for diet, fitness, and home remedies tailored to the predicted disease.
Powered by Gemini API: Generates responses using the Gemini API to enhance user experience.
Prerequisites
Before running the application locally, ensure you have the following installed:

Python 3.x
Streamlit
pandas
scikit-learn
google.generativeai (for Gemini API integration)

Deploying to Streamlit Sharing
Push your code to a GitHub repository.
Log in to Streamlit Sharing (formerly Streamlit Cloud) and create a new app.
Enter the GitHub repository URL and deploy the app.
